# codsoft-recommendation-system
Movie Recommendation System
This project is an AI-based Movie Recommendation System developed using Python, Pandas, Scikit-learn, and Streamlit. It recommends movies similar to a user's selected movie by using a content-based filtering approach. The system analyzes movie features such as genres, keywords, cast, director, and overview, combines them into descriptive tags, and converts them into numerical vectors using CountVectorizer. It then calculates the similarity between movies using Cosine Similarity and recommends the top five most similar movies. The project also includes a simple and interactive Streamlit web interface that allows users to select a movie from a dropdown menu and instantly view personalized recommendations.
